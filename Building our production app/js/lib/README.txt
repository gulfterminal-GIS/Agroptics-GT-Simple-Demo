LEAFLET LIBRARY FILES
=====================

This folder should contain the following files for offline usage:

Required Files:
--------------
1. leaflet.js (from https://leafletjs.com/download.html)
2. leaflet.css
3. leaflet.draw.js (from https://github.com/Leaflet/Leaflet.draw/releases)
4. leaflet.draw.css
5. images/ folder (contains marker and draw tool icons)

Quick Setup:
-----------
1. Open download-libraries.html in the root folder
2. Follow the download links
3. Extract and copy files here

Alternative:
-----------
The application will use CDN links if local files are not found.
This requires an internet connection.

For offline usage, download the files as instructed above.
