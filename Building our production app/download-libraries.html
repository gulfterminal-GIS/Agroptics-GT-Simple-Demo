🌾 GT Agroptics - Download Libraries

Download required Leaflet libraries for offline usage

Step 1: Download Leaflet Core

Download Leaflet.js library (v1.9.4 or later)

📥 Download Leaflet Core

Files needed:

  • leaflet.js
  • leaflet.css
  • images/ folder (marker icons)

Step 2: Download Leaflet.draw

Download Leaflet.draw plugin (v1.0.4 or later)

📥 Download Leaflet.draw

Files needed:

  • leaflet.draw.js
  • leaflet.draw.css
  • images/ folder (draw tool icons)

Step 3: Place Files in Correct Location

Extract and copy all files to js/lib/ folder

js/lib/ ├── leaflet.js ├── leaflet.css ├── leaflet.draw.js ├── leaflet.draw.css └── images/ ├── marker-icon.png ├── marker-shadow.png ├── spritesheet.png └── ... (other icons)
💡 Alternative: Use CDN (Requires Internet)

If you prefer to use CDN instead of local files, the application will automatically fall back to CDN links if local files are not found.

Step 4: Open Application

Once files are in place, simply open index.html in your browser!

✅ Quick Test

After setup, you should see:

  • A satellite map centered on California
  • Drawing tools on the left side
  • Action buttons at the top
